-------------------------------------------------------------------------------
- Web Plugin Update Instructions
-------------------------------------------------------------------------------

1) Pre-installation:
    a) Open Tampermonkey "Options"
    b) Delete any old versions.
    c) Close Chrome completely, then re-open.


2) Installation
    a) Copy and paste this link into your browser:
       https://raw.github.com/dnsev/4cs/master/web/4cs.user.js
    b) Install it using Tampermonkey
