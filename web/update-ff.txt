-------------------------------------------------------------------------------
- Web Plugin Update Instructions
-------------------------------------------------------------------------------

1) Pre-installation:
    a) Go to "Tools" -> "Add-ons" -> "User-Scripts"
    b) "Remove" any older versions.


2) Installation
    a) Copy and paste this link into your browser:
       https://raw.github.com/dnsev/4cs/master/web/4cs.user.js
    b) Install it using Greasemonkey
